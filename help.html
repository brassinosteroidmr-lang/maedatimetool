📚 インフォメーション - 使い方説明書

🎯 このツールについて

インフォメーションは、業務時間管理、定型文管理、出荷データ可視化、カレンダー機能などを統合したオールインワン業務支援ツールです。

⏰ 時計・砂時計エリア

📅 デジタル時計と日付

現在の日付と時刻をリアルタイムで表示します。

⏳ 勤務時間砂時計
  • 開始時刻と終了時刻を設定して「設定」ボタンをクリック
  • 砂時計の砂の量が勤務時間の進捗を視覚的に表示
  • 上部ガラス左側：残り時間
  • 下部ガラス左側：経過時間
⏱️ カウントダウンタイマー
  • 分数を入力して「開始」ボタンをクリック
  • ミニ砂時計が残り時間を視覚的に表示
  • 終了時にアラーム音で通知
  • 「停止」ボタンでリセット可能

📝 定型文機能

よく使う文章を登録して、ワンクリックでクリップボードにコピーできます。

  1. ⚙ボタンをクリックして設定画面を開く
  2. 定型文名と内容を入力
  3. 「追加」ボタンをクリック
  4. 登録した定型文をクリックするとクリップボードにコピー

💼 業務記録機能

📊 業務分析

「📊 分析」リンクをクリックすると、業務記録を分析したダッシュボードを表示できます。

🏷️ クイック業務ボタン
  1. 新しい業務名を入力欄に入力
  2. 「追加」ボタンをクリック
  3. 登録された業務ボタンをクリックで記録開始
  4. もう一度クリックで記録終了
📋 業務履歴

記録された業務は以下の形式で表示されます：

  • 日付（MM/DD）
  • 開始時刻-終了時刻
  • 作業時間
  • 編集・削除ボタン
💾 データ出力
  • 「CSV出力」：業務記録をCSVファイルでダウンロード
  • 「記録削除」：すべての業務記録を削除（定型文やボタンは残る）

📦 出荷データ可視化

📂 CSVファイル読込
  1. デフォルトパスが設定済み（変更可能）
  2. 「📁 ファイル選択」で別のファイルを選択可能
  3. 「📊 データ読込」をクリック
📊 グラフ表示
  • 倉庫ごとにタブで切り替え
  • 3ヶ月分のデータを表示
  • 「◀ 前」「次 ▶」ボタンで期間移動
  • 棒グラフ：数行数
  • 破線：キャパシティ（設定時のみ表示）
⚙ キャパシティ設定
  1. ⚙ボタンをクリック
  2. 倉庫ごとにキャパシティを入力（空欄でキャパなし）
  3. 「💾 設定保存」をクリック

📥 入荷待ちデータ可視化

📂 CSVファイル読込
  1. デフォルトパスが設定済み（変更可能）
  2. 「📁 ファイル選択」で別のファイルを選択可能
  3. 「📊 データ読込」をクリック
⚙ 倉庫マッピング設定

発注納入先住所から倉庫を自動判定するための設定です。

  1. ⚙ボタンをクリック
  2. 倉庫名と判定キーワードを入力
  3. 例：倉庫名「MCUD神戸西倉庫」、キーワード「兵庫県神戸市須磨区弥栄台」
  4. 最大5つの倉庫を設定可能
  5. 「💾 設定保存」をクリック
📊 グラフ表示
  • 倉庫ごとにタブで切り替え
  • 3ヶ月分のデータを表示
  • 「◀ 前」「次 ▶」ボタンで期間移動
  • 棒グラフ：入荷待ち数量
  • 住所情報から自動的に倉庫を判定
倉庫マッピング設定で、住所の一部（例：「兵庫県神戸市須磨区弥栄台」）を指定すると、その文字列を含む住所を自動的に該当倉庫として分類します。

📅 カレンダー機能

🔄 表示モード切り替え

カレンダーの表示内容を3つのモードで切り替えられます：

  • 勤務時間：業務記録がある日に緑ドット表示、日付クリックで詳細表示
  • 出荷データ：選択中の倉庫の出荷数量を各日付に表示
  • 入荷待ち：選択中の倉庫の入荷待ち数量を各日付に表示
🗓️ 6ヶ月カレンダー
  • 今日の日付が強調表示
  • 祝日は赤色で表示
  • 「◀ 前」「今月」「次 ▶」で月移動
📊 カレンダーとグラフの連動
  • 出荷データ・入荷待ちモード時、グラフで倉庫を切り替えるとカレンダーも自動更新
  • CSVデータを読み込むと、即座にカレンダーに反映
  • 数量は青色のバッジで表示
  • データがある日は背景色が変化
📆 納期計算機
  1. 発注日を入力
  2. リードタイム（日数）を入力
  3. 営業日/暦日を選択
  4. 「計算」ボタンをクリック
  5. 納期が表示されます

💾 全設定保存・読込

💾 全設定保存

以下のすべての設定をJSONファイルに保存します：

  • 勤務時間設定
  • クイック業務ボタン
  • 業務記録
  • 定型文設定
  • 倉庫別キャパシティ設定
  • 倉庫マッピング設定
📂 全設定読込

保存したJSONファイルから設定を復元します。複数のPCで同じ設定を使いたい場合に便利です。

定期的に「💾 全設定保存」でバックアップを取っておくことをおすすめします。

🎨 その他の機能

🔔 ブラウザ終了時の警告

業務記録中にブラウザを閉じようとすると、確認ダイアログが表示されます。

💡 使い方のヒント

  • 業務記録は最新20件が表示されます
  • 定型文はクリックするだけで自動コピー
  • 砂時計は視覚的に時間の経過を把握できます
  • すべての設定はブラウザに保存されます（Cookie不使用）
  • 全設定保存機能で他のPCにも設定を移行可能
  • カレンダーのモードを切り替えることで、勤務時間・出荷・入荷を一目で確認
  • グラフとカレンダーが連動するので、倉庫別の詳細な状況を把握しやすい
