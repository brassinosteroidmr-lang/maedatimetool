# 資材課ダッシュボード - カワサキマシンシステムズ

株式会社カワサキマシンシステムズの資材課向けに最適化されたダッシュボードアプリケーションです。

## 機能一覧

### 📰 ニュース表示
- **日経電子版セクション**: 日経電子版へのリンクと案内を表示
- **官公庁情報**: 経済産業省、国土交通省、内閣府の情報へのリンク
  - 製造業政策
  - サプライチェーン対策
  - 物流政策
  - 経済指標

### 📅 カレンダー・時計
- **リアルタイム時計**: 時刻と日付を秒単位で更新
- **月間カレンダー**: 今日の日付をハイライト、前月・次月への移動が可能

### 🔧 資材課専用ツール

#### 1. 在庫管理メモ
- 在庫確認事項をメモとして保存
- ブラウザのローカルストレージに自動保存

#### 2. 発注予定リスト
- 発注予定品目をリスト管理
- 追加・削除機能
- データはローカルに保存

#### 3. 為替レート表示
- USD/JPY（米ドル/円）
- EUR/JPY（ユーロ/円）
- CNY/JPY（人民元/円）
- 30分ごとに自動更新

#### 4. 納期計算機
- 発注日とリードタイム（営業日）を入力
- 土日を除外した納期を自動計算
- 営業日ベースでの正確な納期予測

## 使い方

1. `index.html` をブラウザで開く
2. すべての機能がすぐに利用可能

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: グラデーション、グリッドレイアウト、レスポンシブデザイン
- **JavaScript (Vanilla)**: フレームワーク不要、軽量で高速
- **LocalStorage**: データの永続化（在庫メモ、発注リスト）
- **外部API**: 為替レート取得（exchangerate-api.com）

## カスタマイズポイント

### ニュースソースの追加
`app.js` の `loadGovNews()` 関数にケースを追加

### 為替通貨の追加
`loadExchangeRates()` 関数で追加の通貨レートを計算

### スタイルの変更
`styles.css` でカラースキームやレイアウトを調整

## ブラウザ対応

- Chrome (推奨)
- Edge
- Firefox
- Safari

## 注意事項

- 日経電子版は会員制のため、直接記事を取得できません。ブラウザで別途ログインしてご利用ください。
- 為替レートは参考値です。実際の取引レートとは異なる場合があります。
- データはブラウザのローカルストレージに保存されるため、ブラウザのデータを削除すると消失します。
